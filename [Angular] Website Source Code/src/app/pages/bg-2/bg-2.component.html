PROCEDURAL + FUNCTIONAL + OOP = PYTHON !

Python supported MORE THAN ONE programming styles ! Python source code can be written mainly in three styles: Procedural , Functional , and  Object-Oriented

As any other programming languages, you can write your source in a sequence from top to bottom and Python will execute the code line by line based on the principal of procedural programming . Next, Python also supported some functional programming where you can use the built-in functions like sum, ord, map, and many more. The concepts of object-oriented programming such as classes, inheritance, and encapsulation can implemented in Python.
