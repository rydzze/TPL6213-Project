Selection Statements

In Python, the selection statements, also known as decision control statements or branching statements allow a program to test several conditions and execute instructions based on whether those conditions are True or False. There are two primary selection statements in Python which are ‘if’ and ‘elif’ (short for ‘else if’).

There are two general categories:
a) Two-way selectors
b) Multiple-way selectors
    - if
    - if-else
    - nested if
    - if-elif else
    - nested IF structure
    - nested SWITCH structure
       => dictionary mapping
       => class
